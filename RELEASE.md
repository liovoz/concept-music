## v1.7.0 更新日志

### 🔧 重要修复

- **修复 VIP 领取失败问题** — 解决点击"免费领 1 天畅听 VIP"按钮后提示"网络请求异常"的问题。根因是服务端将酷狗 API 的业务错误码（如"今日已领取"）错误地转换为 HTTP 502 返回给前端，导致前端无法正确识别业务状态。现已将业务错误与网络错误分离，业务错误统一以 HTTP 200 返回，错误码保留在响应体中由前端判断
- **修复 VIP 领取后音质未升级** — 领取 VIP 成功后，当前播放的歌曲会自动升级到最高可用音质（之前仅限于试听歌曲）
- **修复退出登录后状态残留** — 退出登录时现在会完整清理所有用户状态（收藏、VIP、播放历史等），避免切换账号后出现数据混乱

### 🎵 音质解析重构

- **新增高品质音源解析** — 重构音质提取逻辑，正确识别高品质（high）、超清蝰蛇（viper_clear）、全景声（viper_atmos）音质的 hash 值，不再将无损 hash 错误地作为高品质音质的回退
- **新增歌曲 URL 新接口** — 优先使用 `/song/url/new` 接口获取音质映射，支持一次性获取所有可用音质的播放地址，大幅提升音质匹配成功率
- **新增智能音质选择** — 新增 `getBestAvailableQuality` 方法，根据用户 VIP 状态和歌曲可用音质自动选择最优音质
- **优化 URL 缓存策略** — 缓存淘汰时优先清除超过 1 小时的过期条目，而非简单淘汰最旧条目

### 📝 歌词翻译

- **支持歌词翻译显示** — 桌面歌词现在可以显示翻译歌词，优先显示翻译内容，无翻译时回退显示下一行歌词
- **翻译数据多源提取** — 从歌词 API 的 `language` 字段、LRC 文本的 `[language:]` 标签、以及 LRC 格式歌词中多途径提取翻译数据

### 🖥️ 桌面歌词优化

- **修复窗口定位** — 桌面歌词窗口现在会在屏幕中央正确居中显示，不再偏移到屏幕外
- **修复小屏幕适配** — 在屏幕宽度不足时自动调整歌词窗口宽度

### 🛡️ 稳定性改进

- **修复数据遍历死循环** — 在搜索、歌单详情、用户信息等页面的数据遍历中增加 `WeakSet` 循环引用检测，防止 API 返回异常数据时导致页面卡死
- **修复对话框回调重复执行** — 修复 `GlobalDialog` 关闭时回调可能被重复触发的问题，关闭时立即清除回调引用
- **修复搜索频繁触发** — 搜索关键词变化时增加 400ms 防抖，避免路由频繁切换导致请求堆积
- **修复进度条越界** — 拖动进度条时增加边界检查，防止跳转到超出歌曲时长的位置
- **修复预加载音质不匹配** — 下一首歌曲预加载时现在会正确匹配当前音质设置
- **修复 VIP 卡片悬停抖动** — 改进 VIP 卡片显示/隐藏的定时器逻辑，增加防抖处理

### 🔒 网络与错误处理

- **服务端业务错误分离** — `createRequest` 不再将酷狗 API 的业务错误（如已领取、额度上限）转换为 HTTP 502，统一返回 HTTP 200 并在响应体中保留业务错误码
- **前端网络错误识别优化** — `fetchUserInfo` 中的防误杀逻辑不再依赖 HTTP 状态码字符串匹配，改为检测 `network error`、`timeout`、`request failed` 等真正的网络异常特征
- **VIP 领取请求静默化** — VIP 领取相关请求添加 `silent: true`，避免触发全局 `API_GATEWAY_DOWN` 事件导致多余的错误提示
- **VIP 领取网络重试** — 领取 VIP 请求失败时自动重试一次，并从错误响应中提取业务错误码进行判断

### 🪟 其他改进

- **后端启动失败提示** — 本地 API 服务未能在规定时间内启动时，弹出错误提示框告知用户
- **弹窗键盘支持** — 退出登录确认弹窗和 VIP 升级弹窗支持 Escape 键关闭，打开时自动聚焦
- **图片无障碍属性** — 为用户头像、歌单封面、歌曲封面等图片添加 `alt` 属性
