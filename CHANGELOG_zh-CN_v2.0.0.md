# v2.0.0 更新说明（简体中文）

v2.0.0 是一个里程碑版本。我们围绕播放器体验、页面交互、稳定性与发布规范进行了系统升级，并以此版本作为后续迭代的新基础。

## 重点更新

### 播放器与交互体验升级
- 优化播放器状态联动与播放流程，提升播放稳定性。
- 完善歌曲操作交互，增强常用操作的可达性与一致性。
- 优化全局弹窗与交互细节，减少打断感与误触发。

### 页面体验全面优化
- 优化发现页、搜索页、排行榜详情、歌单详情、专辑详情、歌手详情、每日推荐、新歌、历史记录等页面的交互与展示。
- 桌面歌词相关体验继续打磨，显示逻辑与可读性进一步提升。
- 统一头部、侧边栏、底部播放器栏的行为细节与体验节奏。

### 稳定性与工程能力增强
- 优化用户状态与播放器状态管理逻辑，降低异常场景下的不一致问题。
- 改进请求与错误处理流程，提升复杂网络环境下的可用性。
- 持续优化 Electron 主进程与预加载通信逻辑，增强桌面端运行可靠性。

### 仓库与发布基线整理
- 清理旧版发布文档与历史无关内容，仓库结构更清晰。
- 规范发布流程与产物结构，方便用户下载与升级。
- v2.0.0 作为后续版本统一基线。

## 安装与更新

- 新用户：下载 `Setup` 安装包完成安装。
- 已安装用户：可在应用内执行检查更新（以客户端更新配置为准）。

如遇问题或有建议，欢迎通过 GitHub Issues 反馈，我们会持续跟进优化。
