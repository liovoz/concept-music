# 概念音乐 v3.0.0

这是一次面向桌面端体验、播放稳定性和发布链路整理的重新发布版本。建议先删除旧 Releases，再以 `v3.0.0` 作为新的首个可下载版本重新发布。

## 主要更新

- 全新整理桌面端项目文档，README 已按当前 Electron + Vue 3 项目结构重新编写。
- 优化播放体验，支持播放队列、播放模式切换、桌面歌词、系统托盘控制和自动更新检查。
- 支持多音质播放与自动音质选择，包含标准、HQ、SQ、Hi-Res、超清蝰蛇、全景声等音质入口。
- 新增/完善“播放全部”渐进加载逻辑，歌单、榜单、搜索结果等长列表可以更平滑地加入播放队列。
- 完善私人 FM 推荐体验，支持当前 FM 队列、相似推荐、喜欢/不喜欢、高潮片段播放等交互。
- 完善歌曲、歌手、专辑、歌单、排行榜、每日推荐、新歌速递、听歌足迹等页面的跳转与播放能力。
- 增加音量增强能力，支持本地音频代理与多档增益设置。
- 内置本地 KuGou API 服务，桌面应用启动时自动拉起 `127.0.0.1:10420` 服务。
- 收紧仓库忽略规则，避免依赖目录、构建产物、浏览器临时配置、缓存、测试报告和敏感凭据文件被误上传。

## 下载

请在本 Release 的 Assets 中下载：

- `概念音乐 Setup 3.0.0.exe`：Windows 安装版，推荐普通用户使用。
- `概念音乐-3.0.0-Portable.exe`：Windows 便携版，无需安装，下载后直接运行。

## 安装说明

1. 下载适合你的 Windows 版本安装包。
2. 如果使用安装版，运行 `概念音乐 Setup 3.0.0.exe` 并按提示完成安装。
3. 如果使用便携版，直接运行 `概念音乐-3.0.0-Portable.exe`。
4. 首次启动后可登录账号以使用每日推荐、我的歌单、喜欢列表、听歌足迹等能力。

## 注意事项

- 本项目仅用于学习、研究和个人技术交流，请尊重音乐版权和平台规则。
- 部分音乐、音质或账号能力依赖第三方接口及账号状态，实际可用性可能受平台策略影响。
- 如果自动更新不可用，可回到 GitHub Releases 手动下载安装新版。

## 校验建议

发布前建议确认：

- `package.json` 版本已改为 `3.0.0`。
- `package-lock.json` 根版本已同步为 `3.0.0`。
- `npm run build` 可以通过。
- GitHub Actions 的 `Build & Release` 工作流可以成功生成安装版和便携版。
