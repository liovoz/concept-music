# 概念音乐 Desktop

一款轻量、干净、即开即用的桌面音乐播放器，基于 Electron + Vue 3 构建。

## 功能特性

- 🎵 高品质音乐播放
- 📝 桌面歌词显示
- 🔄 智能播放列表
- 📱 优雅的用户界面
- 🔔 自动更新提醒

## 技术栈

- **框架**: Vue 3 + Vite
- **UI**: TailwindCSS 3
- **状态管理**: Pinia
- **桌面端**: Electron 30
- **构建工具**: electron-builder

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 启动 Electron
npm start
```

## 打包

```bash
# 构建
npm run build

# 打包发布
npm run pack
```
