# 概念音乐 Desktop

一款轻量、干净、即开即用的桌面音乐播放器，基于 Electron + Vue 3 构建。

## 功能特性

### 核心播放
- 🎵 高品质音乐播放
- 📝 桌面歌词显示
- 🔄 智能播放列表（顺序/随机/单曲循环）
- ⏯️ 完整的播放控制（上一首/下一首/暂停/进度条）

### 私人FM ✨
- 🎲 **个性化推荐**：基于算法的智能音乐推荐
- 🚫 **不喜欢标记**：跳过不喜欢的歌曲，优化推荐结果
- ♾️ **无限播放**：持续推荐，无需手动选歌
- 🔀 **一键切换**：随时进入/退出私人FM模式

### 用户功能
- 👤 用户登录与信息展示
- 📜 播放历史记录
- ❤️ 收藏与管理

### 系统特性
- 📱 优雅的用户界面（暗色主题）
- 🔔 自动更新提醒
- ⚡ 快速响应与流畅体验

## 技术栈

- **框架**: Vue 3 + Vite
- **UI**: TailwindCSS 3
- **状态管理**: Pinia
- **桌面端**: Electron 30
- **构建工具**: electron-builder

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 启动 Electron
npm start
```

## 打包

```bash
# 构建
npm run build

# 打包发布
npm run pack
```
